1. What is an Algorithm?

An algorithm is a step-by-step procedure to solve a problem. It must be finite, unambiguous, and effective.

%%{init: {'theme': 'dark'}}%% graph LR A[INPUT] --> B[ALGORITHM
Step-by-step rules] B --> C[OUTPUT] style A fill:#0369a1,stroke:#06b6d4,color:#fff style B fill:#075985,stroke:#38bdf8,color:#fff style C fill:#0369a1,stroke:#06b6d4,color:#fff

Example: Sum of two numbers

Step 1: Start
Step 2: Input a, b
Step 3: sum = a + b
Step 4: Print sum
Step 5: Stop

Pseudocode

BEGIN
    READ a, b
    sum ← a + b
    DISPLAY sum
END
