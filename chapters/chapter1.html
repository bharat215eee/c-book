1. What is an Algorithm?

An algorithm is a set of instructions for solving a problem or accomplishing a task.

For designing any solution, the following must be known:

  • Input – Data to be processed
  • Tasks to be performed – Steps to transform input
  • Output expected – Final result

2. Flowcharts

Flowcharts are the graphical representation of a solution to a particular problem which comes under the category of Programming Practices and Techniques.

In simple words, a flowchart is a graphical representation of a program.

  • Symbolic diagrams showing data type, data flow, control flow, and logic
  • Uses pictorial representation instead of descriptive steps (algorithm)
