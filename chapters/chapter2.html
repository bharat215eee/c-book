Chapter 2: C Tokens

Understanding the smallest building blocks of C programs

Adapted from Dr. N Bharath Kumar ‚Äî Unit: C Tokens

1. What are C Tokens?

A token is the smallest meaningful unit of a C program that the compiler recognizes. The compiler breaks the source code into tokens during lexical analysis.

%%{init: {'theme': 'dark'}}%% graph TD T[Tokens] --> A[Keywords] T --> B[Identifiers] T --> C[Constants] T --> D[Strings/Characters] T --> E[Operators] T --> F[Punctuations] T --> G[White Spaces / Comments] style T fill:#0369a1,stroke:#06b6d4,color:#fff style A fill:#075985,stroke:#38bdf8,color:#fff style B fill:#0f172a,stroke:#60a5fa,color:#fff style C fill:#0369a1,stroke:#7dd3fc,color:#fff

Tokens include: keywords, identifiers, constants, string/character literals, operators, and punctuation symbols.

3. Keywords

Keywords are reserved words with special meaning in C. They cannot be used as identifiers. Standard C has 32 keywords (e.g., int, return, if, while).

4. Identifiers & Variables

Identifiers are names you give to variables, functions, arrays, etc. Variables map to memory locations and can be local, global, static or external.

%%{init: {'theme': 'dark'}}%% graph TD A[Identifier] --> B[Variable] B --> C[Local] B --> D[Global] B --> E[Static] B --> F[External] style A fill:#0369a1,stroke:#06b6d4,color:#fff style C fill:#075985,stroke:#38bdf8,color:#fff style D fill:#0f172a,stroke:#60a5fa,color:#fff

Naming rules

  • Start with a letter or underscore.
  • Contain only letters, digits, and underscore.
  • Not a keyword; case-sensitive.
  • Up to 31 significant characters (historical).

Memory mapping example

int x = 10;  // stored at some memory address
float pi = 3.14; // another address
// Variables occupy memory locations identified by names (identifiers)

7. Type Modifiers & Qualifiers

Modifiers alter the meaning/size of base types; qualifiers change variable properties like mutability.

Type Modifiers

  • short / long ‚Äî change storage size of integers
  • signed / unsigned ‚Äî with/without sign

Qualifiers

  • const ‚Äî read-only variable
  • volatile ‚Äî value may change unexpectedly (hardware/ISR)

8. Punctuations & Operators

Punctuation characters like { } ( ) ; , # organize the program. Operators perform arithmetic, comparison and logical operations.

Operator groups

  • Arithmetic: + - * / %
  • Relational: == != > < >= <=
  • Logical: && || !
  • Bitwise: & | ^ ~ << >>
  • Assignment: = += -= *= /=
